# Best UTCS Machine
A utility to find the best lab machine with lowest load time and number of current users


## Installation:
    1. Install the dependencies
    2. Run main.py

## Requirements:
    Python 3

## Dependencies:
    BeautifulSoup4 for Python3
    `pip3 install beautifulsoup4` to install
